# dalnoebshiki2
Расширение под chrome, для олдскульного перемещения по google street view
![overlay](https://github.com/artesk/dalnoebshiki2/assets/1773067/3cdf693b-2305-4db5-96cd-e996f227320f)


*Шаги для тестирования расширения*:
1. Откройте Chrome и перейдите на chrome://extensions/.
2. Включите режим разработчика в правом верхнем углу.
3. Нажмите "Загрузить распакованное расширение" и выберите папку с вашим проектом.
4. Перейдите на Google Maps и войдите в режим Street View, чтобы увидеть оверлей кабины грузовика.

## Обновление HUD

Верхняя панель построена на HTML, CSS и угловатых SVG-глифах в стиле почтового индекса. Ее значения можно менять без пересоздания оверлея через событие `dalnoboyshiki2:hud-update`:

```js
window.dispatchEvent(
  new CustomEvent("dalnoboyshiki2:hud-update", {
    detail: {
      gear: 4,
      rpm: 3150,
      fuelPercent: 12,
      engineWarning: true,
    },
  }),
);
```

`fuelPercent` принимает значения от 0 до 100, а `rpm` — реальные обороты двигателя. Панель сама преобразует обороты в формат `x100` и включает предупреждение при остатке топлива 15% или меньше.

## Телеметрия Street View

Счётчик `ВРЕМЯ` запускается при входе в Street View и сбрасывается после выхода. Координаты текущей панорамы считываются из URL Google Maps каждые 500 мс.

При изменении координат расширение:

- считает переход как один ход;
- вычисляет длину перехода по координатам и прибавляет её к общему пути;
- рассчитывает скорость по длине перехода и времени между соседними координатами;
- обнуляет скорость после двух секунд без движения;
- не добавляет в пробег скачки длиннее 500 метров, считая их переносом в другую точку.

Путь и количество переходов отображаются в правой секции HUD. Значения `time` и `speedKmh` по-прежнему можно передать через событие для отладки, но автоматическая телеметрия перезапишет их при следующем обновлении.

При превышении 100 км/ч справа появляется игровое уведомление о фиксации нарушения и штрафе. Оно показывается пять секунд и сможет сработать повторно после снижения скорости до 95 км/ч.

## Шрифт

Для подписей HUD используется локальная копия [Press Start 2P](https://github.com/google/fonts/tree/main/ofl/pressstart2p). Шрифт распространяется по лицензии SIL Open Font License 1.1; текст лицензии находится в `fonts/OFL-PressStart2P.txt`.

## Фирменная музыка

В режиме Street View в правой части приборной панели появляется магнитола `FM RADIO`. По нажатию она открывает компактный встроенный YouTube-плеер с [плейлистом саундтрека](https://www.youtube.com/watch?v=x2vnaAdm-Rg&list=PL50DEA6B792AFF6BE).

Плеер загружается только после нажатия пользователя. Кнопка `ВЫКЛ` закрывает панель, удаляет iframe и останавливает воспроизведение. Если автор конкретного ролика запретил встраивание, под плеером остаётся ссылка для открытия плейлиста непосредственно на YouTube.
